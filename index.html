Sobre mi

Mi nombre es Iván, nací en Argentina, en la provincia de Buenos Aires, y actualmente vivo en zona norte, Tigre. Soy técnico electromecánico y programador por pasión. En este momento, estoy cursando la carrera de Técnico Programador en la UTN (Universidad Tecnológica Nacional). Trabajar como programador se ha convertido en uno de mis mayores anhelos, ya que he descubierto en la programación algo que no encontré en otras áreas. Además, realicé un curso de tester en la Fundación Carlos Slim, lo que me permitió adquirir conocimientos en la prueba de aplicaciones y mejorar mis habilidades técnicas.

Conosimientos

Soy un programador con conocimientos en Programación Orientada a Objetos (POO), bases de datos, y manipulación de archivos. Domino lenguajes como C++, C#, SQL y .NET, además de tener experiencia en el desarrollo de aplicaciones utilizando WinForms, un framework de .NET para crear interfaces gráficas. También tengo una base sólida en tecnologías web, incluyendo HTML, CSS y JavaScript, lo que me permite desarrollar sitios y aplicaciones web con funcionalidad tanto en el frontend como en el backend.

Proyectos

Goblin´s Gold Hunt

Mi juego, Goblin's Gold Hunt, es un juego en 2D con gráficos en estilo pixel art. El jugador controla a un duende que debe recolectar la mayor cantidad de monedas de oro posible antes de perder sus tres vidas. Está ambientado en un bosque donde tendrás que enfrentarte a diferentes animales, como ardillas y búhos, mientras evitas sus ataques. El juego registra una puntuación basada en las monedas recolectadas. Lo he desarrollado usando el IDE Visual Studio Community, empleando la biblioteca SFML y programado en C++.

Programa de gestion de articulos

He trabajado en el desarrollo de un programa diseñado para gestionar información de artículos en una base de datos alojada en un servidor. Esta base de datos, construida en SQL, almacena detalles como el nombre del artículo, su descripción, marca, categoría, precio y múltiples imágenes asociadas. El programa incluye diversas funcionalidades, como la posibilidad de agregar nuevos artículos al catálogo, listar los artículos ya cargados, y modificar cualquier información relacionada. También permite añadir nuevas categorías o marcas, gestionar imágenes adicionales para los productos, y eliminar aquellos artículos cuya información ya no sea necesaria. Este programa está diseñado para el sistema operativo Windows, desarrollado en el IDE Visual Studio Community utilizando el lenguaje C# y con una base de datos en Azure Management (SQL). La interfaz gráfica fue creada utilizando el framework .NET.

Este portfolio

He creado este portfolio utilizando el IDE Visual Studio Code, empleando tecnologías web como HTML, CSS y JavaScript. El proyecto destaca por su diseño elegante y funcionalidad dinámica, con una estructura bien definida que facilita la navegación. El HTML se encarga de la estructura básica del sitio, mientras que el CSS se utiliza para aplicar estilos visuales y crear una experiencia atractiva para el usuario. El JavaScript agrega interactividad, permitiendo una experiencia más envolvente y personalizada. Este enfoque integral asegura que el portfolio no solo sea visualmente atractivo, sino también funcional y accesible.
